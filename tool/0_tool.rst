###################
mobilgene A Studio
###################


This section describes how to use mobilgene A Studio that is Adaptive AUTOSAR modeling tool. 
It provides users who design and deploy platform for target ECU with modeling approach and configuration methods.

This section includes the concept and overview, installation and running, and use of functions of mobilgene A Studio.
It also describes how to create and configure Adaptive Application, Service, Machine, and Functional Clusters in compliance with the Adaptive AUTOSAR specification.
